 # Bookstore API (FastAPI + PostgreSQL)
 Проект реализует API для управления книжным магазином.
 ## Как запустить:
 1. Активируйте виртуальное окружение: `source venv/bin/activate`
 2. Установите зависимости: `pip install -r requirements.txt`
 3. Убедитесь, что PostgreSQL запущен и создана база `octagon_db`.
 4. Запустите сервер: `uvicorn app.main:app --reload`
 5. Перейдите по адресу: [http://127.0.0.1:8000/docs]
